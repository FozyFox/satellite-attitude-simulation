# satellite-attitude-simulation
A Python simulation of spacecraft attitude dynamics using quaternions and PD control.

Nothing much. Just run the program, and/or change the q_d value to change the desired quaternion value.


I personally only worked on the quaternion math. The plotting, 1 or 2 of the functions, and the idea was by @donutgoose. He also taught me a bit about how to implement the quaternion math.
